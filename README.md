# Credit Scoring Production

Déploiement en production du modèle de scoring développé dans le projet OpenClassrooms MLOps.

## Objectifs

- API FastAPI
- Déploiement Docker
- CI/CD GitHub Actions
- Déploiement Render
- Stockage des prédictions avec Neon PostgreSQL
- Monitoring avec Streamlit
- Détection de Data Drift avec Evidently AI
